About HTML

  • HTML: Hypertext Markup Language

  • What is it used for?:

    • "Allows the user to create and structure sections, paragraphs, headings, links, and blockquotes for web pages and applications."

  • Advantages:

    • "Straightforward and easy to learn even for complete beginners."

    • "Every browser supports HTML Language."
  • Disadvantages:

    • "HTML can create only static and plain pages so if we need dynamic pages then HTML is not useful."

    • "Security features are not good at HTML."
  • History:

    • "HTML was invented by Tim Berners-Lee, a physicist at the CERN research institute in Switzerland."
